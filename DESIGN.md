# Design System - Allianz

## Typography
- Font Family: `'Allianz Neo', -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif`
- Headings: Bold (`--core-font-weight-200` / `700`) or Semi-Bold (`--core-font-weight-150` / `600`) using Allianz Blue `#003781`.
- Sizing:
  - Display headers: `1.75rem` (`28px`)
  - Page titles: `1.5rem` (`24px`)
  - Subheaders / Card titles: `1.15rem` (`18px`)
  - Body text: `0.95rem` (`15px`)
  - Muted labels / Captions: `0.85rem` (`13px`)

## Colors
- Main Background: `#F5F7FA`
- Surface Background (Cards, forms): `#FFFFFF`
- Borders: `1px solid rgba(0, 55, 129, 0.1)`
- Primary Accent: `#003781`
- Secondary Accent / Hover: `#00266D`
- Active State / Selected: `#001955`
- Success: `#05813C` (green-700)
- Error: `#DD0A42` (red-700)
- Warning: `#FAB600` (yellow-400)

## Layout & Components
- Shadows: Subtle box-shadow to elevate surfaces slightly. Avoid harsh black dropshadows.
  - Standard Surface: `0 4px 12px rgba(0, 55, 129, 0.05), 0 1px 3px rgba(0, 55, 129, 0.03)`
  - Active/Hover: `0 8px 24px rgba(0, 55, 129, 0.08), 0 2px 6px rgba(0, 55, 129, 0.05)`
- Border Radius:
  - Cards: `12px`
  - Form Fields & Buttons: `8px`
- Transitions:
  - Duration: `0.2s`
  - Easing: `cubic-bezier(0.4, 0, 0.2, 1)` (smooth ease-in-out)
